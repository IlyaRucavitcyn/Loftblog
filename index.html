Немного обо мне

Я живу в Минске (Беларусь). Мне 30 лет. По образованию я инженер-энергетик, окончил БНТУ в 2009 году. Опыт в разработке - 1.5 года. Всем приобретенным навыкам обязан курсам на htmlacademy.ru, прекрасному учебнику по JS, а также моим дорогим друзьям-программистам. На текущий момент самостоятельно зарестайлил сайт психологической помощи.

Мой опыт в разработке

Достаточно уверенно владею HTML5 и CSS3. Есть опыт работы с Tweeter Bootstrap и Google MDL. Решаю основные задачи на JS с использованием jQuery и без. Знаком с Backbone.js, MongoDB. Есть опыт full-stack разработки для Node.js с использованием express.js. Уверенно использую Git в своем workflow. Активно пользуюсь gulp при разработке. Дебажу в Chrome с использованием DevTools и Node-Inspector. Активно изучаю TDD :)
