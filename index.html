Достаточно уверенно владею HTML5 и CSS3. Есть опыт работы с Tweeter Bootstrap и Google MDL. Решаю основные задачи на JS с использованием jQuery и без. Знаком с Backbone.js, MongoDB. Есть опыт full-stack разработки для Node.js с использованием express.js. Уверенно использую Git в своем workflow. Активно пользуюсь gulp при разработке.
